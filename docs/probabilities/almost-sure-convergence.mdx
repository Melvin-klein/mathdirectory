# Almost sure convergence

:::info
This section is waiting for [contributions](/contributing).
:::

Soient $(X_n), X$ des v.a. définies sur le même espace probabilisé $(\Omega, A, P)$ à valeurs dans $\mathbb R$ (ou $\mathbb R^d$).

## Definition

:::definition
We say that $X_n$ converges almost surely to $X$ if $\mathbb P(X_n \rightarrow X) = 1$
:::

## Results

:::proposition
If $X_n$ converges to $X$ almost surely then $X_n$ converges in probability to $X$.
:::
