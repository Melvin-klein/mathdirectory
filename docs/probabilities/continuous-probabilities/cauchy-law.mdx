# Cauchy's law

## Definition

:::definition
Let $c > 0$. $X$ follows the Cauchy's law with the parameter $c$ if its density is :

$$
f_X(x) = \frac{1}{\pi}\frac{c}{c^2+x^2}.
$$

We note $X \sim \mathcal{C}(c)$.
:::
