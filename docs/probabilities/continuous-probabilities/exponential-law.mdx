# Exponential law

## Definition

:::definition
Let $\lambda > 0$. $X$ follows the exponential law with parameter $\lambda$ if its density is :

$$
f_X(x) = \lambda e^{-\lambda x}\mathbb{1}_{\mathbb{R}^+}(x).
$$

We note $X \sim \mathcal{E}(\lambda)$
:::
